# 1project Tools Guide

## Preferred Tool Order
- read_file + run_in_terminal (git branch --show-current) before artifact updates
- grep_search for existing project IDs and branch references
- apply_patch for project artifact edits

## Anti-patterns
- Do not invent project IDs or branch names.
- Do not hand off with missing Branch Plan.

## Notes
- Keep actions scoped to project branch and allowed files.
- Prefer deterministic commands with evidence-producing output.
