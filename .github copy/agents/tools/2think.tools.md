# 2think Tools Guide

## Preferred Tool Order
- search_subagent + read_file for discovery
- run_in_terminal for quick repository evidence commands
- apply_patch for think artifact updates

## Anti-patterns
- Do not rewrite code or tests.
- Do not skip branch-gate verification evidence.

## Notes
- Keep actions scoped to project branch and allowed files.
- Prefer deterministic commands with evidence-producing output.
