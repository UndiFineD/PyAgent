## Implementation

### `src/context_manager/__init__.py`
- `ContextManager(max_tokens)` — async context store with token-budget pruning
- `push(text)` — async append with automatic oldest-segment pruning
- `snapshot()` — return current context as a single string

### Design Principles
- **Token-budget pruning**: segments are dropped from the oldest end when the
  whitespace-tokenised word count exceeds `max_tokens`
- **Immutable history model**: each `push` creates a new state; old segments
  are not mutated, only removed from the live window
- **Async-first**: `push` is a coroutine to allow future I/O integration
