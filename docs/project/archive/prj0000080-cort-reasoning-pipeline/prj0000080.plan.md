## Overview

Implements the **Chain-of-Recursive-Thoughts (CoRT)** reasoning pipeline as a
reusable async module under `src/core/reasoning/`. The design (Option A тАФ Fixed
N rounds ├Ч M alternatives, E2 heuristic EvaluationEngine) is fully specified in
`prj0000080.design.md`. This plan breaks that design into four TDD sprints that
produce real, importable, test-passing code with no stubs or skeletons.

**4 sprints тЖТ 7 source files + 3 test files**

| Sprint | Deliverable | Tasks |
|--------|-------------|-------|
| S1 | Data models + LlmCallable protocol | T1тАУT4 |
| S2 | EvaluationEngine | T5тАУT6 |
| S3 | CortCore loop | T7тАУT10 |
| S4 | CortAgent + CortMixin | T11тАУT13 |

**Hard constraints carried from design:**
- `n_rounds ├Ч m_alternatives тЙд 15` enforced by `CortConfig.__post_init__`
- `asyncio.gather` used for per-round parallel LLM calls
- `ContextTransaction` (outer stable ID + inner round IDs) guards re-entry
- `EvaluationEngine` is pure-stdlib in V1 тАФ zero extra LLM calls
- All files use PascalCase names and carry the Apache 2.0 license header

- [ ] **T8** тАФ Implement `CortCore._run_round`
  - **Files:** `src/core/reasoning/CortCore.py` (edit)
  - **Acceptance criteria:**
    - Opens inner `ContextTransaction(f"cort:{agent_id}:round:{round_n}")` as async context manager
    - Calls `_build_prompt` тЖТ calls `_generate_alternatives` тЖТ scores each chain with `evaluator.score()` тЖТ `dataclasses.replace(chain, score=rubric.weighted_total)` тЖТ calls `evaluator.select_best(scored_chains)` тЖТ returns `CortRound`
    - `CortRound.prompt_used` equals the output of `_build_prompt`
    - `CortRound.alternatives` is a `tuple` of all scored chains
    - `CortRound.winner` is the chain returned by `select_best`

- [ ] **T9** тАФ Implement `CortCore.run`
  - **Files:** `src/core/reasoning/CortCore.py` (edit)
  - **Acceptance criteria:**
    - `time.monotonic()` captured at entry; `elapsed_seconds = time.monotonic() - start`
    - Initial `best_chain = ReasoningChain(text="", score=0.0, round_n=-1, temperature=0.0, alternative_idx=0)`
    - Outer loop calls `_run_round(prompt, best_chain, round_n)` for `round_n in range(config.n_rounds)`
    - `best_chain` updated to `round_result.winner` after each round
    - `rounds_history` accumulated as a list, converted to `tuple` for `CortResult`
    - `CortMetadata` assembled with `agent_id=""` (CortCore has no agent_id; caller sets it), `rounds_executed=len(rounds_history)`, `total_llm_calls=len(rounds_history) * config.m_alternatives` (approximation; actual successful calls counted in later refinement), `elapsed_seconds`
    - Returns `CortResult(best_chain, tuple(rounds_history), metadata)`
