## Implementation

### `src/chat/models.py`
- `ChatRoom` — in-memory chat room with `post(sender, text)` and `history()` methods

### `src/chat/utils.py`
- `create_personal_room(human)` — creates `personal-{human}` room with user + agent-id participant

### `src/chat/api.py`
- FastAPI app with `/chat/rooms` (POST), `/chat/rooms/{room_id}/messages` (POST/GET) endpoints
- Prometheus counter for message metrics (fallback stub for minimal envs)
- `MemoryTransaction` integration for atomic room state updates

### `src/chat/mcp_tools.py`
- MCP-compatible tool definitions for chat operations

### `src/chat/streaming.py`
- `word_chunks(text, delay)` — async generator yielding words with optional delay (demo/test helper)
- `collect(stream)` — consume an async text stream and return the full concatenated string
- `stream_to_sse(stream, event, done_event)` — wrap any async text stream as SSE events (JSON payload per token, sequential `id:` fields, terminal `[DONE]` event)
- `StreamingChatSession(room_name)` — stateful session accumulating chunk-by-chunk streaming with `full_text`, `is_finished`, `to_dict()` and `clear()`
